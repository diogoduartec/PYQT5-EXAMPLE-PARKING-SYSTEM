# Parking

### run virtual environment Ubuntu:

```shell
source venv/bin/activate
```
### run app:

```shell
python3 main.py
```
## Technologies

##### 1. [Python3](https://docs.python.org/3.3/)
##### 2. [PyQt5](https://pypi.org/project/PyQt5/)
##### 3. [Sqlite3](https://www.sqlite.org/index.html)
