# Parking

### Setup virtual environment Ubuntu:
Install Poetry https://python-poetry.org/docs/configuration/

Init poetry virtual environment console
```shell
poetry shell
```

Install dependencies
```shell
poetry install
```
### run app:

### Run the project

```shell
python main.py
```
